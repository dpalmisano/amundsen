# amundsen

A collection of projects related to (sub)-marine environment simulations.

## Overview

This repository serves as a monorepo for simulation tools and models focused on submarine and marine environments. It is named after [Roald Amundsen](https://en.wikipedia.org/wiki/Roald_Amundsen), the Norwegian explorer renowned for his polar and maritime expeditions.

## Contents

This repository will contain:

- **Simulation models** – computational models for simulating submarine and marine physical processes (e.g. fluid dynamics, pressure, temperature gradients).
- **Data utilities** – tools for ingesting, processing, and visualising oceanographic and environmental data.
- **Scenario configurations** – pre-defined scenario setups for common (sub)-marine simulation use cases.
- **Documentation** – guides and references to help contributors and users understand and extend the simulations.

## Getting Started

More detailed setup and usage instructions will be added as individual projects are contributed to this repository. Stay tuned!
