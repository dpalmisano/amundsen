# amundsen

A collection of projects related to (sub)-marine environment simulations.

## Overview

This repository serves as a monorepo for simulation tools and models focused on submarine and marine environments. It is named after [Roald Amundsen](https://en.wikipedia.org/wiki/Roald_Amundsen), the Norwegian explorer renowned for his polar and maritime expeditions.
